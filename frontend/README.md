This is a [Next.js](https://nextjs.org) for the Until Web app.

## Setup project

```bash
pnpm install
cp .env.example .env
```

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Deploy on Vercel

```bash
  vercel deploy --prod
```
